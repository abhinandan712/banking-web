# Backend Deployment

Deploy this backend folder separately on Vercel:

1. Go to vercel.com
2. Click "Add New Project"
3. Import your GitHub repo: `abhinandan712/banking-web`
4. Set **Root Directory** to `backend`
5. Click Deploy

After deployment, you'll get a backend URL like:
`https://banking-web-backend-xxx.vercel.app`

Then update the frontend to use this URL.
